Keyboard Ghosting & NKRO

Ghosting usually shows up when you press several keys at once (common in games). This guide explains what it is and how to check it quickly.

What is “ghosting”?

  • Ghosting: the keyboard reports an extra key you did not press, or it fails to report a key when multiple keys are held.
  • Rollover (KRO): how many keys can be reliably registered at the same time.
  • NKRO: “n-key rollover” — the keyboard can register many keys simultaneously (often over USB with a proper implementation).

How to test ghosting (simple)

  1. Open the Keyboard Tester Online.
  2. Hold a common cluster like W + A + Shift and then tap Space (or try your favorite in-game combination).
  3. Watch which keys light up. Try adding 1–2 more keys and see when a key stops registering.
Tip: For a repeatable test, try grid combos like QWE + ASD or number-row keys.

Why it happens

Many non-mechanical or older keyboards use a matrix wiring layout. Some combinations share the same electrical rows/columns, which can create conflicts when pressed together.

What to do if you see ghosting

  • If it’s a laptop keyboard, ghosting is often a hardware limitation.
  • Try another USB port or remove hubs (rarely helps, but easy to test).
  • For gaming, consider a keyboard advertised as “anti-ghosting” or “NKRO.”
