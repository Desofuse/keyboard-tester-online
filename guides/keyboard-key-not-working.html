Keyboard key not working — what to do

This checklist helps you quickly figure out whether the problem is software, dirt, a loose switch, or the keyboard itself.

1) Confirm the key is actually dead

  • Open the tester and press the key several times.
  • Try the key with Shift, Fn, and Caps Lock (some keys behave differently).
  • If you have a laptop: try with and without external keyboard, and test in a different app (browser + notes app).

2) Quick software checks

  • Restart (yes, really). Stuck keyboard drivers happen.
  • Try a different browser. If it works elsewhere, the issue may be an extension or browser setting.
  • On Windows: check language / layout (ENG vs. another layout) and accessibility settings (Sticky Keys, Filter Keys).
  • On Mac: check input source and any keyboard remapping tools.

3) Physical checks (most common)

  • Turn the keyboard upside down and gently tap to remove debris.
  • Use compressed air around the problem key.
  • If it is a mechanical keyboard: remove the keycap and inspect for hair/dust.
  • If a key is sticky, carefully clean with isopropyl alcohol (avoid soaking).

If the keyboard is under warranty, avoid deep disassembly. Do the basic cleaning only.

4) Hardware isolation

  • Test on a different computer. If the key is dead on multiple machines, it is hardware.
  • Try a different USB port or cable (for wired keyboards).
  • For Bluetooth keyboards: re-pair the device and test with fresh batteries/charge.

5) If only some keys fail together

It may be a matrix/ghosting limitation or a stuck modifier. If multiple keys fail when pressed together, read:
